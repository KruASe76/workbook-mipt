## Геометрические вероятности

**Важные моменты:**
- независимые СВ
- равномерное распределение СВ
- конечное пространство исходов

Это метод решения задач на теорию вероятностей, который используется, когда множество исходов удобно представить в виде части координатной плоскости.  
Обычно это так, когда принадлежность исхода рассматриваемому событию задается функцией-предикатом от двух **независимых** случайных величин с **равномерным распределением** - тогда этим величинам сопоставляются оси координатной плоскости, общее множество исходов представляется в виде **конечной** области (исходя из ограничений параметров), а множество исходов события - в виде внутренней области (исходя из функции-предиката).  
В таком случае, вероятностью события будет попросту отношение площадей областей (или иной меры, определенной на области).

**Классные примеры:**
- два человека приходят в случайный момент в течение часа, и каждый из них ждет полчаса: встретятся ли?
- стержень ломают в двух случайных местах: получится ли треугольник?

## Дисперсия линейной комбинации
$$\mathbb{D} \left( \sum\limits_{i=1}^{n} \alpha_{i}\xi_{i} \right) = \sum\limits_{i=1}^{n} \sum\limits_{j=1}^{n} \alpha_{i} \alpha_{j} \text{cov}(\xi_{i}, \xi_{j}) = \sum\limits_{i=1}^{n} \alpha_{i}^{2} \mathbb{D}\xi_{i} \ + \ 2 \sum\limits_{1 \le i \lt j \le n} \alpha_{i} \alpha_{j} \text{cov}(\xi_{i}, \xi_{j})$$
$$\text{cov}(\alpha_{1}\xi_{1}, \alpha_{2}\xi_{2}) = \mathbb{E} \left( \ (\alpha_{1}\xi_{1} - \mathbb{E}(\alpha_{1}\xi_{1})) \ (\alpha_{2}\xi_{2} - \mathbb{E}(\alpha_{2}\xi_{2})) \ \right) = \alpha_{1}\alpha_{2} \ \mathbb{E} \left( \ (\xi_{1} - \mathbb{E}(\xi_{1})) \ (\xi_{2} - \mathbb{E}(\xi_{2})) \ \right) = \alpha_{1}\alpha_{2} \text{cov}(\xi_{1}, \xi_{2})$$
